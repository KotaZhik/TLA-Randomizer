# Randomizer for That Level Again 1 & 2 (PC Only)

Randomizer is a mod for the That Level Again series (TLA1 and TLA2) that completely shuffles the level order.
Every time you complete a level or touch a teleport, the game loads a random level!

## Installation (TLA 1)

1. **Download the mod:** Get the latest `Randomizer.dll` from the [Releases page](https://github.com/KotaZhik/TLA-Randomizer/releases).
2. Download **BepInEx 5.4+** from the official website: [Download](https://github.com/BepInEx/BepInEx/releases)
3. Unpack the archive and copy the **BepInEx** folder into the `That Level Again` game folder
4. Launch the game, then close it. This will create the necessary folders
5. A **plugins** folder will appear inside `That Level Again\BepInEx`
6. Copy the **`Randomizer.dll`** file into `That Level Again\BepInEx\plugins`
7. Launch the game and the mod will activate automatically!

## Installation (TLA 2)

For That Level Again 2, the installation is identical. Just copy the BepInEx folder into the TLA2 game folder and follow the same steps.
